--class selector

/* All elements with class="spacious" */
.spacious {
  margin: 20px;
}

/* All <li> elements with class="spacious" */
li.spacious {
  margin: 20px;
}

li, .spacious {

----------------------------------------------------- 3- CSS Media Queries

eg- only apply this css when the screen is x-pixels

Can also hide things on mobile devices
